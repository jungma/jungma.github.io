<br />
1. HTML내용 데이터바인딩<br />
2. HTML속성 데이터바인딩<br />

4. HTML을 for문 돌려서 넣기<br />

6. 역방향 데이터바인딩 (input 등에 글자를 넣으면 자바스크립트 데이터가 바뀐다능)<br />

### 기본문법
``` javascript
function multiply(a, b = 1) {
  return a*b;
}

multiply(5, 2); // 10
multiply(5); // 5
multiply(5, undefined); // 5
```
